# 马前卒人物 Fact 卡片（首轮整理）

本文件是基于仓库现有公开材料整理的人物 fact 清单。

目标不是堆八卦，而是整理可回溯、可分级、可用于后续 persona 补强的公开事实。

当前版本优先收录三类内容：

- 履历与自我叙述
- 童年经历与稳定表达
- 公开文化偏好与内容习惯

## 使用规则

- `verified`：有较明确的一手或接近一手材料支持，表述尽量保守
- `candidate`：有明显线索，但不宜直接上升为稳定人格标签
- `avoid`：虽然公开出现过，但要么敏感，要么极容易被误用
