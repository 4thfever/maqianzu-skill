# 开发档案

本目录用于存放本仓库的开发过程文档、研究笔记与阶段性计划。

这些文件不属于首次使用仓库时的必读内容，但对于理解仓库是如何设计、研究、实现和逐步整理出来的会很有帮助。

## 包含内容

- [execution-plan.md](execution-plan.md)：项目执行计划与阶段拆分
- [research-source-log.md](research-source-log.md)：外部资料摸底与来源记录
- [research-style-observations.md](research-style-observations.md)：风格观察笔记
- [research-analysis-framework.md](research-analysis-framework.md)：分析框架研究笔记
- [research-persona.md](research-persona.md)：persona 研究笔记
- [person-fact-workflow.md](person-fact-workflow.md)：人物 fact 检索与证据分级模板
- [maqianzu-person-facts.md](maqianzu-person-facts.md)：已整理的马前卒人物 fact 卡片
- [maqianzu-person-facts-index.md](maqianzu-person-facts-index.md)：人物 fact 的轻量检索入口
- [source-layout.md](source-layout.md)：上游语料目录形态观察

## 使用建议

- 想直接使用仓库：优先阅读顶层 `README.md`、`AGENTS.md`、`GETTING_STARTED.md`
- 想理解仓库如何构建：阅读 `docs/build-process.md`、`docs/architecture.md`、`docs/maintenance.md`
- 想追溯开发过程：从本目录开始

## 说明

- 本目录中的部分文件保留了项目早期阶段的措辞，例如 “skill” 等表述，主要用于记录当时的设计背景。
- 如与当前仓库主路径描述不一致，以顶层 `README.md`、`AGENTS.md` 和 `GETTING_STARTED.md` 为准。
