# 马前卒资料来源日志

## 目的

本文件用于承接 Phase 2A 的外部资料摸底结果。

目标不是立刻给出“马前卒是什么风格”的最终结论，而是先回答三个问题：

1. 互联网上有哪些可用来源
2. 这些来源分别属于哪一类
3. 哪些来源值得在 Phase 2B 继续深入阅读和提炼

## 分类规则

为便于后续整理，本日志按四类来源记录：

- 第一手公开材料：优先级最高，可直接用于后续风格和方法论提炼
- 第二手整理材料：帮助建立时间线、栏目结构和入口，不直接作为最终结论依据
- 外界评价材料：用于识别公众感知中的风格标签和争议点
- 节目外补充材料：用于观察其在不同场景中的表达是否稳定

### 可信度标记

- `高`：第一手材料或结构清晰、可追溯的来源
- `中`：有参考价值，但需要和第一手材料交叉验证
- `低`：只能作为外界观感样本，不能直接进入 prompt 结论

### 当前状态标记

- `已发现`：已纳入来源池，尚未细读
- `待精读`：已确认值得后续重点阅读
- `低优先`：可作边缘补充，不作为主要投入对象

## 五、Phase 2A 初步观察

基于当前摸底，可以先得到几条方法上的判断，但这些判断暂时不进入正式 prompt：

- 第一手材料入口并不缺，后续关键不是“有没有材料”，而是“怎么筛和怎么分层”。
- 节目本体之外，微博、直播、幕后访谈可以帮助分辨哪些风格是节目体裁造成的，哪些是其相对稳定的个人表达。
- 第二手整理材料对建立索引很有帮助，但很容易混入外界标签，后续必须和第一手节目材料对照。
- 外界评价对识别争议点很有价值，尤其能帮助我们避免把单一立场下的夸赞或批评误写成稳定人格设定。

## 六、建议的下一步精读顺序

Phase 2B 建议优先精读以下来源：

1. `bedtimenews-archive-contents` 中的 `main` 与 `livestream`
2. 观察者网马前卒工作室的节目文字页
3. 睡前消息编辑部微博
4. “答案茶话会04”及其他幕后/访谈类材料
5. 节目索引 Gist
6. 维基百科与其他二手整理页
7. 外界评价材料

## 七、暂不直接写入 prompt 的内容

以下内容即使在互联网上很常见，也不应在 Phase 2A 后立刻写入正式 prompt：

- 带强烈褒贬色彩的人设标签
- 明显来源于某一场争议事件的短期印象
- 无法在第一手材料中反复观察到的“风格结论”
- 对其本人身份、动机或私下立场的推测
