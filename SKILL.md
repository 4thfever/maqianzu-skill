# Compatibility Note

本仓库的主要 OpenClaw skill 入口已经迁移到：

- `skills/maqianzu/SKILL.md`

如果你的运行环境仍然从仓库根目录查找 `SKILL.md`，请把它视为兼容说明，而不是主入口文件。

OpenClaw 主路径应配合以下文件使用：

- `AGENTS.md`
- `SOUL.md`
- `TOOLS.md`
- `skills/maqianzu/SKILL.md`
